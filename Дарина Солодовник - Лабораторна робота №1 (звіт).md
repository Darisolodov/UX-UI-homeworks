**Гіпервізор** — це софт, який дозволяє одному фізичному комп’ютеру запускати кілька операційних систем одночасно. Він розподіляє ресурси заліза (процесор, пам'ять, диск) між цими системами так, щоб вони не заважали одна одній.

**I тип (Bare-metal):** Встановлюється безпосередньо на «залізо». Він швидкий і стабільний, бо йому не потрібна основна ОС для роботи.

**II тип (Hosted):** Працює як звичайна програма всередині вже встановленої операційної системи (наприклад, Windows або macOS). Зручно для тестів, але повільніше через зайвий шар керування.
